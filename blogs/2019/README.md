## 自我勉励
* 这世上所有的不公平都是因为当事人能力的不足
* 有时成长需要一些强制性！自己要给自己做汇报与记录！
* 要学会相信一些更为长远的东西，而不是只一味看短期和眼前的回报!
* 只要功夫深，铁杵磨成针！只有想不到，没有做不到！
* 承前启后，既往开来!

* 好习惯有四：准时，正确，恒心，迅速。缺少第一项，光阴会虚度；不具备第二项，错误百出；没有第三项，事情永远办不好；丢失第四项，遇上良机，都会白白错失。
